About

ThootTrack is an app designed to help small dental clinics manage their appointments and patients more efficiently. It was inspired by the need for a digital solution in the dental industry. As a dental student working on my doctorate, I saw the challenges faced by clinics still using outdated methods like Excel. This project aims to provide a simple yet powerful tool for managing dental clinics.

This project is part of my Portfolio for Holberton School. You can find the source code on GitHub.
